Lights Out

Toggle all the lights off to win!

Total Clicks: 0

Addendum

This game is designed to replicate the classic *Lights Out* puzzle, with an interactive twist. The key features of the game include:

  • Random Starting Configuration: The puzzle board is generated with a random, solvable configuration each time the game starts, ensuring that every game is a new challenge.
  • Interactive Design: Each tile on the board is clickable, and clicking a tile toggles it between on and off. The state of neighboring tiles is also toggled, which adds a layer of strategy to the game.
  • Click Counter: Keep track of your progress! The number of clicks you take to win the game is displayed in the corner of the screen.
  • Win Condition: The objective is to turn off all the lights on the board. Once all lights are off, a "You win!" message is displayed.
  • User Experience Enhancements: The game uses smooth animations, intuitive controls, and visually appealing colors to make the experience enjoyable.

Design Choices:

The game features a 5x5 grid of tiles, each of which can be toggled between two states: on (represented by a red light) and off (represented by a green light). The puzzle is designed to be solvable within a reasonable number of moves.

The board is randomized each time you play, but it is always solvable, so you can always complete the puzzle. The game is designed for short sessions, so it's ideal for casual gameplay.
